Planning Questions

Answer these to shape the product and site.

1) What problem does 1 Link Me solve?

People often want to share many links at once. 1 Link Me combines them into a single shareable link.

2) How does the “one link” work today?

The single link contains your list of links inside it. When someone opens that link, the page reads the data and shows the list. This means there is no server needed right now.

3) Do we store the user’s links somewhere?

Right now, the links are not stored on a server. They live inside the shareable URL itself. If you lose the URL, there is no way to recover the list. This is simple and private, but it does not allow long-term storage unless the user saves the URL.

4) Should we store links for the future?

This is a product decision. Storing links on a server makes it easy for users to retrieve their lists later and allows features like analytics or editing, but it requires a backend, a database, and a privacy policy that covers stored data.

5) What is the maximum number of links?

Free users can add up to 8 links. A future paid plan can allow more than 8.

6) What happens when someone opens a shared link?

They see a clean list of the links and can click each one. There is an "Edit these links" button that returns to the builder.

7) Is there any sign-in?

No, the current version has no accounts.

8) Can we add ads?

Yes, but ad platforms typically require compliance pages (Privacy Policy / Terms) and may require user consent banners for some regions.

9) Future ideas (optional)

Custom short domains (like 1linkme.co), link analytics, user accounts, custom link previews, and saved link collections.
